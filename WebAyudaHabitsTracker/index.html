Introducción

habitsTracker es una aplicación diseñada para ayudarte a formar y mantener hábitos. Con una interfaz intuitiva, podrás registrar tus hábitos diarios y visualizar tu progreso a través de gráficos claros además de tener un sistema de rachas para mantener la iniciativa.

Instalación

  1. Descarga la app desde el repositorio de Github.
  2. Instala Android Studio (recomendado: Android Studio Meerkat Feature Drop | 2024.3.2 Patch 1).
  3. Instala MySQL localmente o usa Docker.
  4. Ejecuta el script inicial de la base de datos.
  5. Abre el proyecto en Android Studio y ejecuta el emulador.

Uso

Registra una nueva cuenta

  1. Al abrir la app sin sesión, pulsa en "¿No tienes cuenta? Regístrate".
  2. Introduce nombre, correo y contraseña.
  3. Opcional: sube una imagen de perfil o usa la generada automáticamente.

Crear un nuevo hábito

  1. Pulsa el botón "+" en la pantalla principal.
  2. Introduce nombre y descripción del hábito.
  3. Guarda para comenzar a registrar.

Completa un hábito

Haz clic en la checkbox para marcar el hábito como completado.

Edita un hábito

  1. Arrastra el hábito hacia la derecha.
  2. Modifica los campos en la pantalla de edición.
  3. Guarda para aplicar cambios.

Elimina un hábito

Arrastra el hábito hacia la izquierda para eliminarlo.

Comprueba tus estadísticas

Pulsa el icono de estadísticas en el menú lateral o junto al botón "+".

Funcionalidades

  • Registro diario de hábitos.
  • Edición de hábitos.
  • Gráficos semanales.
  • Sistema de rachas.
  • Borrado de hábitos.
  • Creación de cuenta.
  • Imagen de perfil.
  • Imagen de perfil generada automáticamente.
